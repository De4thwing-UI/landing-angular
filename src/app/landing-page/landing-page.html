У 2025 році стартує вибір електронних версій оригінал-макетів підручників для учнів з навчанням українською мовою та мовами національних меншин.

На цій сторінці ви знайдете інформацію про наші підручники та етапи проведення конкурсу НУШ 2025. Долучайтеся та обирайте найкраще для своїх учнів.

Конкурс підручників НУШ 2025: як відбувається відбір?

Підручники для 1–3 класів

Кожна школа обирає один комплект з кількох запропонованих варіантів для кожного предмета.

  • Перегляд електронних оригінал-макетів.
  • Обговорення в педагогічній спільноті.
  • Вибір фаворита для замовлення друкованих примірників.

Підручники для 4 класу

Відбір триває у два етапи, щоб школи отримали найкраще рішення.

  • Первинне рейтингове оцінювання та обрання трьох варіантів.
  • Фінальне голосування за один комплект для друку.
  • Підготовка до забезпечення учнів друкованими підручниками.

Що пропонують підручники НУШ від видавництва «Ранок»?

Електронні освітні додатки

Усі підручники НУШ «Ранок» мають інтерактивні додатки, які допомагають зробити уроки цікавішими та сучаснішими.

  • Слухай: аудіо, мелодії, пісні, диктанти.
  • Дивись: анімації, відео, презентації, вебсторінки.
  • Вивчай: ілюстрації, завдання, карти, комікси.
  • Грай: інтерактивні ігри та завдання.

Методична підтримка

Учителі отримують готові матеріали й поради для проведення уроків та виконання вимог НУШ.

  • Розробки уроків, календарне планування, інструктажі.
  • Методичні рекомендації з оцінювання навчальних досягнень.
  • Сценарії тематичних і позакласних заходів.

Навчальні й методичні заходи

Спільно з авторами проводимо зустрічі та вебінари, щоб підтримати педагогів у роботі з нашими підручниками.

  • Навчальні зустрічі, консультації, воркшопи.
  • Авторські вебінари з рекомендаціями щодо використання підручників.

Як взяти участь у виборі підручників НУШ 2025

Для вчителів

Перегляньте наші електронні макети, обговоріть їх з колегами та рекомендуйте комплект, який відповідає потребам вашого класу.

Для адміністрації

Здійсніть офіційний вибір на рівні закладу освіти та оформіть замовлення на друковані підручники для учнів.
